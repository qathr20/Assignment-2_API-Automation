API testing merupakan jenis pengujian software yang melibatkan API secara langsung dan sebagai bagian dari pengujian integrasi untuk menentukan apakah memenuhi fungsionalitas, reliabiliti, performa, dan keamanan.

Object Repository -> menyimpan semua endpoint Web service dengan semua informasirequest method, URL, header, content, dan authentication.
Test Case -> menyimpan semua test skenario dengan profil eksekusi yang ditentukan.
